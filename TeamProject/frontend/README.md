## Project setup

### Requirement

- Yarn 
  - Download link: https://classic.yarnpkg.com/en/docs/install/#windows-stable

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn run serve
```

### Compiles and minifies for production

```
yarn run build
```

### Run your tests

```
yarn run test
```

### Lints and fixes files

```
yarn run lint
```
